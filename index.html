SauNova

An AI-powered smart sauna companion that combines retrieval-augmented generation (RAG) with personalized machine learning recommendations. SauNova provides intelligent coaching by grounding responses in sauna science research while delivering customized temperature, humidity, and session length recommendations based on user physiology and goals.

The platform features RAG chat capabilities that draw from scientific PDFs for evidence-based guidance, a PyTorch neural network for personalized recommendations, and comprehensive session telemetry with dual-axis visualizations and AI-generated briefings. Built with a FastAPI backend, Flutter mobile app, and TypeScript bridge for seamless integration.

Technologies: Python, FastAPI, PyTorch, LangChain, OpenAI GPT-4o-mini, FAISS, Flutter, TypeScript, Node.js, scikit-learn, pandas, NumPy, matplotlib, HuggingFace Transformers
